Selector Component

Introduction

The selector is a DropDown component with a few simple goals

  • Data bound - Easy to bind to an array or observable source
  • Styleable - Replaceable and customizeable CSS classes for everything
  • Filterable - Let the user search available items
  • Lazy loading - Show X items and load more items if requested
  • Reactive Forms - Works with reactive forms
  • Mobile - should be useable on mobile devices
  • Multi select - use should be able to select multiple items
